# 源代码 使用手册



## 注意

- 如果是__github__下载的要在客户端或者服务端的__项目目录下__执行命令：

  npm i 

  以下载相关依赖



## 客服端

- 位置：./client
- 开启命令：npm run dev



## 服务端（模拟后端）

#### 注意：服务端仅为调试网络传输代码时使用

- 位置：./server
- 开启命令:node index.js
